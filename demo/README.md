## 系統入口

### 01 - 登入頁面

![登入頁面](01-登入頁面.png)

系統登入介面，採用深色主題設計，以公司 NAS SMB 帳號進行身份驗證。

### 02 - 桌面概覽

![桌面概覽](02-桌面概覽.png)

類桌面操作介面，頂部為系統列（時間、通知、使用者），中央為應用程式圖示區，底部為常用功能 Dock。

應用程式包含：
- 檔案管理、終端機、VSCode
- 專案管理、AI 助手
- Prompt 編輯器、Agent 設定、AI Log
- 知識庫、Line Bot、分享管理、系統設定

### 03 - 訊息中心

![訊息中心](03-訊息中心.png)

系統訊息中心，顯示：
- 登入成功/失敗記錄
- 安全相關通知
- 支援按程度、來源篩選
- 分頁瀏覽歷史訊息

### 04 - 使用者資訊

![使用者資訊](04-使用者資訊.png)

點擊右上角使用者名稱開啟，顯示：
- 使用者頭像與名稱
- 顯示名稱設定
- 首次登入時間
- 最後登入時間

## Line Bot 管理

### 09 - 分享管理

![分享管理](09-分享管理.png)

管理所有已建立的分享連結：
- 顯示分享類型（專案/知識庫/NAS 檔案）
- 建立者與建立時間
- 有效期限與存取次數
- 可複製連結、開啟或撤銷

### 10 - Line Bot - 我的綁定

![LineBot-我的綁定](10-LineBot-我的綁定.png)

顯示當前使用者的 Line 帳號綁定狀態：
- 綁定的 Line 名稱
- 綁定時間
- 解除綁定按鈕

### 11 - Line Bot - 群組

![LineBot-群組](11-LineBot-群組.png)

Line Bot 群組管理頁面：
- 群組列表（名稱、成員數、狀態）
- AI 回應開關
- 群組詳情（專案綁定、最近訊息）
- 支援分頁瀏覽

### 12 - Line Bot - 訊息

![LineBot-訊息](12-LineBot-訊息.png)

Line Bot 訊息記錄查詢：
- 依對話/群組篩選
- 顯示 Bot 回應與使用者訊息
- 時間戳記
- 支援分頁瀏覽

### 13 - Line Bot - 檔案

![LineBot-檔案](13-LineBot-檔案.png)

Line Bot 接收的檔案管理：
- 依群組、檔案類型篩選

## AI 功能

### 15 - AI Log

![AI-Log](15-AI-Log.png)

AI 對話記錄查詢系統：
- 依 Agent、類型、狀態篩選
- 顯示成功率、平均耗時、Token 統計
- 工具呼叫列表（MCP 工具使用記錄）
- Log 詳情：System Prompt、使用者輸入、執行流程、最終回應

### 16 - Agent 設定

![Agent設定](16-Agent設定.png)

AI Agent 配置管理：
- Agent 列表（Line 群組、Line 個人助理、系統排程、程式碼助手等）
- 基本資訊（名稱、說明）
- AI 設定（Model 選擇、System Prompt）
- 工具權限（網路搜尋、網頁抓取、檔案讀寫等）
- 啟用/停用狀態
- 測試 Agent 功能

### 17 - Prompt 編輯器

![Prompt編輯器](17-Prompt編輯器.png)

System Prompt 管理工具：
- Prompt 列表（依類型分類：System、Task、Template）
- 名稱、顯示名稱、分類設定
- Prompt 內容編輯
- 最後更新時間
- 儲存與刪除功能

### 18 - AI 助手

![AI助手](18-AI助手.png)

AI 對話介面：
- 左側：對話列表管理
- Agent 選擇（網路搜尋、預設對話等）
- 模型顯示（Claude Sonnet）
- Token 使用量顯示
- Markdown 渲染的回應內容
- 串流式回應

## 技術特色

- **類桌面體驗**：視窗管理、拖曳調整、Window Snap
- **深色/亮色主題**：CSS Variables 設計系統
- **即時通訊**：Socket.IO 終端機、AI 串流回應
- **AI 整合**：Claude API、MCP 工具、多 Agent 支援
- **Line Bot**：訊息記錄、AI 回應、知識庫整合
- **NAS 整合**：SMB 檔案存取、預覽、分享
